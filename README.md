## Personalities

| Vibe | Description |
|------|-------------|
| **Espresso Addict** | Intense, ambitious, probably vibrating |
| **Latte Art Master** | Aesthetic, cozy, main character energy |
| **Chill Cold Brew** | Efficient, unbothered, low-maintenance |
| **Chaos Matcha** | Trendy, chaotic, spiritually caffeinated |

## Features

| Feature | Description |
|---------|-------------|
| **8 Questions** | Personality-driven quiz flow |
| **Viral Sharing** | Twitter, TikTok, copy link |
| **Email Capture** | Zapier webhook integration |
| **Analytics** | Vercel Analytics built-in |
| **OG Images** | Social sharing optimized |

## Quick Start

```bash
git clone https://github.com/DeadManOfficial/coffee-vibe-quiz.git
cd coffee-vibe-quiz
npm install
npm run dev
```

---

## Deploy

```bash
# One-click deploy
./deploy.sh

# Or manual
npm i -g vercel && vercel --prod
```
